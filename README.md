# destination-pike
The development and deployment environment for destinationpike.com

This site is generated by wordpress using the simplystatic plugin. I hope to eventually automate that process so you will commit a wordpress exported settings file and we during deployment we will create a wordpress instance, export that data and then upload it as a static site to cloudflare.

For the time being a docker file to stand up wordpress is included and the exported content is included and auto-deployed to cloudflare
